Port and Vulnerability Scanning Tool

This project is a modular and high-performance Port Scanning and Vulnerability Scanning tool, designed to scan all ports on a target system. It supports TCP, UDP, and SYN scans and provides a framework for checking for potential vulnerabilities based on the detected open ports and services. The project is optimized for speed, utilizing multi-threading and other performance enhancements, and is built with a clean and modular architecture.

Enigma Machine

This project is a Python-simulated version of the legendary Enigma machine used during World War II, designed to encrypt and decrypt messages through a user-friendly web interface. It replicates the historical encryption mechanism, allowing users to securely encode and decode messages with the same logic employed during wartime. The project is built with a clean and modular architecture, ensuring ease of use and extensibility for educational or experimental purposes.

Malware Insight

Malware Insight is a Python-based tool designed for detecting malicious content in files through advanced techniques such as file signature checks, YARA rule analysis, and VirusTotal API integration. Aimed at cybersecurity professionals and researchers, it provides a comprehensive solution for thorough malware analysis, built with a modular and high-performance architecture to ensure efficiency and scalability.

Megapot Honeypot

This project is a Python-based honeypot system that utilizes TCP/IP protocol tunneling to simulate ports such as FTP, Telnet, SSH, SMTP, and HTTP, actively operating in live environments. Designed to detect and analyze cyber attacks, it features a modular and high-performance architecture for efficient threat monitoring and analysis.
